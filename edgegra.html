EdgeGra

About

EdgeGra allows you to create custom wallpapers with gradients, detects your device and preloads some settings that can be changed at any time.

With EdgeGra you can:

  • Create wallpapers with gradient borders
  • Create wallpapers with gradient background
  • Create wallpapers with your own image and set custom blur
  • Create wallpapers with notch, dock or widgets
  • You can configure thickness, blur and also set dashed borders
